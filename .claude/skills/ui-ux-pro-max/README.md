# UI-UX Pro Max Skill

**Source:** [nextlevelbuilder/ui-ux-pro-max-skill](https://github.com/nextlevelbuilder/ui-ux-pro-max-skill)
**Version:** 2.5.0 | **License:** MIT

## What It Provides

AI-powered design intelligence with:
- **67 UI styles** — from minimalism to cyberpunk aesthetics
- **161 color palettes** — aligned to product categories and industries
- **57 Google Fonts pairings** — curated heading + body combinations
- **25 chart type recommendations** — data visualization guidance
- **99 UX guidelines** — best practices across domains
- **161 industry-specific reasoning rules** — for domain-aware design decisions

## Usage in This Project

Invoke via the `/design` or `/ui-review` commands.

### When to Use

- Choosing color palettes for the grocery storefront branding
- Selecting font pairings for the store's typography
- Designing new banner layouts or product card styles
- Reviewing existing UI for UX best practices
- Making responsive design decisions for mobile-first layouts

### E-Commerce/Grocery Relevance

This skill is particularly useful for:
- **Food/grocery vertical** color psychology (freshness, appetite, trust)
- **Product grid layouts** for browsing and discovery
- **Checkout flow UX** — reducing cart abandonment
- **Mobile shopping patterns** — thumb-friendly zones, quick-add buttons
- **Accessibility** — color contrast ratios, readable fonts at small sizes

## Integration

The skill's knowledge is embedded in the `/design` and `/ui-review` command files. It uses the skill's reasoning rules and databases to generate context-aware design recommendations for this specific project.
